# 🎯 HOW TO USE ARIA - SIMPLE GUIDE

## ✅ STEP 1: Start the Website

1. Open **Terminal** (black window)
2. Copy and paste this:
   ```
   cd ~/slack-aria-hr-bot && node dashboard-server.js
   ```
3. Press **Enter**
4. You'll see "Dashboard is running!"
5. **Keep this window open** - don't close it

---

## ✅ STEP 2: Open the Website

1. Open **Chrome** or **Safari**
2. Type this in address bar: `localhost:3001`
3. Press **Enter**
4. You'll see the Aria AI homepage with 4 big buttons

---

## 🎨 WHAT YOU'LL SEE

### Main Page (Home)
Four big cards with buttons:

1. **📊 Hiring Dashboard** - Click to see all your hiring numbers
2. **🎯 Candidate Sourcer** - Click to find candidates
3. **✅ Resume Screener** - Click to check resumes
4. **💬 Interview Generator** - Click to make interview questions

---

## 🛠️ HOW TO USE EACH TOOL

### 1. 📊 HIRING DASHBOARD
- **What it does:** Shows all your hiring data
- **How to use:**
  1. Click "View Dashboard" button
  2. See all 15 open positions
  3. See charts and numbers
  4. That's it! Just view the data

---

### 2. 🎯 CANDIDATE SOURCER
- **What it does:** Creates search strings to find candidates
- **How to use:**
  1. Click "Find Candidates" button
  2. You'll see a big text box
  3. Copy-paste your job description OR just type the role name
  4. Click "Generate Sourcing Strategy" button
  5. Wait 10-20 seconds (AI is working)
  6. You'll get search strings to use on LinkedIn/Naukri

**Example:** Just type "Senior Backend Engineer" and click the button!

---

### 3. ✅ RESUME SCREENER
- **What it does:** Reads resumes and scores them 0-100
- **How to use:**
  1. Click "Screen Resumes" button
  2. You'll see TWO text boxes
  3. **First box:** Paste job description
  4. **Second box:** Paste candidate's resume
     - Open resume PDF → Select All → Copy
     - Or copy from LinkedIn/Naukri profile
  5. Click "Analyze Resume" button
  6. Wait 10-20 seconds
  7. You'll get a score (0-100) and recommendation

---

### 4. 💬 INTERVIEW GENERATOR
- **What it does:** Creates 20-30 interview questions
- **How to use:**
  1. Click "Generate Questions" button
  2. You'll see a text box and a dropdown
  3. **Text box:** Paste job description
  4. **Dropdown:** Choose interview type
     - Comprehensive = All types of questions
     - Technical = Only tech questions
     - Behavioral = Soft skills questions
     - Leadership = For manager roles
  5. Click "Generate Interview Questions"
  6. Wait 15-30 seconds
  7. You'll get 20-30 questions ready to use

---

## 💡 SIMPLE TIPS

### ✅ DO THIS:
- Keep the Terminal window open while using
- Use Chrome or Safari browser
- Copy-paste job descriptions from your files
- Copy-paste resumes from PDFs (Select All → Copy)
- Wait patiently when it says "AI is working..."

### ❌ DON'T DO THIS:
- Don't close the Terminal window (website will stop working)
- Don't close browser while AI is generating (wait for it to finish)
- Don't worry if it takes 10-30 seconds (that's normal)

---

## 🔄 HOW TO STOP

When you're done:
1. Go to Terminal window
2. Press **Ctrl + C** (Control and C together)
3. Website will stop

To start again later:
1. Open Terminal
2. Run the same command: `cd ~/slack-aria-hr-bot && node dashboard-server.js`
3. Open browser to `localhost:3001`

---

## 🆘 IF SOMETHING DOESN'T WORK

### Problem: "Can't connect" or "Page not found"
**Solution:** Make sure Terminal window is still open and running

### Problem: "Error" message appears
**Solution:**
1. Press Ctrl+C in Terminal
2. Run the command again
3. Try again in browser

### Problem: Takes forever to load
**Solution:**
1. Check your internet (AI needs internet to work)
2. Wait up to 30 seconds - sometimes it's just slow
3. If still not working, refresh the page
